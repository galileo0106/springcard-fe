The ID Only Template is the basic Template to read a card or tag's protocol-level IDentifier. Pay attention that such IDs may easily be cloned or spoofed, and that NFC mobile phones as well as modern, privacy-concerned card solutions, use random protocol-level IDentifiers, that would change all the time.
