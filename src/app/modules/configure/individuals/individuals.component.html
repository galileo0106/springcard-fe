License cryptogram

This register is available to store a proof that the Application name and License key are genuine.

Integrators and solution providers are welcomed to create a bundle tying the product to an accompanying software. This register may store a proof that the product has been personalized by the provider of the software. An easy way to do so is to compute an HMAC over the product's serial number, the Application Name and the Application License. The application may verify the HMAC (provided that it knows the secret key used to compute it). This register has a size of 32 bytes, which allows to use HMAC-SHA256.

 label License cryptogramUse hexadecimal notation (i.e FA04D3)

Pay attention that this register may normally be updated by the user. Set appropriate values in the Access Conditions register to protect it against unwanted changes.
