Key ID key for Master Card and Config push

Master Cards are protected by a key-set, combining AES keys to read the Master Card, and an ECC key-pair to prove the authenticity of its content. This register stores the Key ID that identifies the key-set. The same Key ID is used for Config push.

Master Cards are protected by a key-set, combining AES keys to read the Master Card, and an ECC key-pair to prove the authenticity of its content. This register stores the AES key that protects the data on the card. The same AES key is used for Config push.
